# Task Distribution System

## Amaç
Analistin verdiği **Zorluk** seviyesine göre görevleri:
- adil dağıtmak (aynı zorluktaki yük dengeli),
- ve **ardışık zorluk (±1) yasağına** uymak.

## Teknoloji
- .NET 7, ASP.NET Core MVC
- EF Core 3.1 (SQLite)
- C#

## Kurulum
```bash
dotnet build
dotnet ef database update
dotnet run
```
